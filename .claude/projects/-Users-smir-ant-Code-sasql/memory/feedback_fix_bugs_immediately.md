---
name: Fix bugs immediately, never defer
description: User's universal rule — fix bugs the moment they are found, never accumulate technical debt
type: feedback
---

Always fix bugs immediately when found. Never defer them, never accumulate them, never say "fix later."

**Why:** The user considers deferred bugs as accumulated technical debt. Every unfixed bug is a risk that compounds. Asking "fix now or move on?" is the wrong question — the answer is always "fix now."

**How to apply:** This applies to ALL projects, not just sasql. When an audit or test reveals a bug, fix it in the same session before moving to new features. Do not create "fix later" tasks. Do not ask the user whether to fix — just fix.
